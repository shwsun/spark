# Oozie install in docker  
```bash
cd oozie 
docker build -t shwsun/oozie .

docker run -itd --privileged --name oozie --hostname oozie --rm -p 10000-10001:11000-11001 shwsun/oozie

docker exec -it oozie jps

```
